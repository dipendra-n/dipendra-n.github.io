Seeksha: Skill development for youth through learning from others

System Design

  1. Portfolio
  2. Seeksha: Skill development for youth through learning from others

Brief
The project was carried out in a team of 2 as part of the academic course, ‘Systems Approach to Design’. From the domains listed by the instructor, we picked the domain ‘Skill Development for youth using Technology’. We started by identifying some statistics regarding the percentage of youth in India, the unskilled youth and the internet penetration in various cities in India. We identified a gap in terms of knowledge level between the skilled and the unskilled youth of India. Taking this as a problem area, we aimed to create a platform that connects users to learn skills from other users. A self-sustaining community which results in a healthy flow of knowledge and information aimed at skill development. We looked upon various existing solutions and identified the shortcomings of the existing systems like lack of personalization in the learning. We decided to create a social platform, which would allow learners to search for teachers, get in touch with them, chat/ have a video call with them or meet them in person. To make the system engaging, we introduced gamification though ‘Karma Points’, which are reward points for teachers and ‘Badges’. We created a data flow diagram, information architecture, followed by low fidelity prototype using Axure RP and then User Interface mockups with a friendly visual language. The project was a great learning experience which taught me about designing a system with multiple users and being able to visualize how data flows and get stored in a system.
