About
As part of the 2 month summer Internship at Fractal Ink, I worked primarily on Windows 8 applications and some website designs. Fractal Ink has won various awards from Kyoorius, Confederation of Indian Industry (CII) and has been recognized as India’s best UX Design Studio by the Pool Magazine.

Process
My work involved in brainstorming, creating user journeys, information architecture, low fidelity wireframing and rendering high fidelity GUI mockups. The user interfaces designed for the Windows 8 apps were based upon Microsoft’s Metro UI guidelines and usability for touch interfaces was taken into consideration.
The projects followed a Waterfall development model and each stage was followed by getting client feedback, co ordinating with engineers and iterating within strict time frames.
Tools used primarily included Adobe Photoshop, Illustrator.
